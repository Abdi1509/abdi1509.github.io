Portfolio-prosjekt

Dette prosjektet er en moderne portefølje laget med HTML, CSS og React. Responsivt design og animasjoner gjør det både funksjonelt og estetisk.

HTML CSS React

Problemstilling

Hvordan lage en moderne portefølje som viser ferdigheter, prosjekter og interaktivitet, samtidig som den er responsiv og brukervennlig.

Min rolle i prosessen

Jeg designet hele frontend-strukturen, implementerte React-komponenter, CSS-animajsoner og interaktiv Snake-spill integrasjon.

Resultat

En moderne, responsiv porteføljeside med dynamisk innhold, interaktive elementer og språkvalg. Prosjektet viser både design- og utviklingsferdigheter.
